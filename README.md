# TransitFlow EJS Project

## Run Project

```bash
npm install
npm start
```

Open this URL in browser:

```txt
http://localhost:3000
```

## Development Run

```bash
npm run dev
```

## Routes

```txt
/
/about
/services
/project
/contact
```
